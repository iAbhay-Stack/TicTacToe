# Tic Tac Toe Game

## Overview

Tic Tac Toe is a simple console-based game developed in C++. The game allows two players to take turns marking spaces on a 3×3 grid. The first player to align three of their symbols horizontally, vertically, or diagonally wins the game.

This project demonstrates the use of arrays, loops, conditional statements, functions, and game logic in C++.


---

## Game Rules

1. The game is played on a 3×3 board.
2. Player X starts first.
3. Players take turns entering a position from 1 to 9.
4. The first player to get three marks in a row, column, or diagonal wins.
5. If all positions are filled and no player wins, the game ends in a draw.
